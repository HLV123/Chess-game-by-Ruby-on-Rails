Web game cờ vua — SolidJS + Ruby on Rails + PostgreSQL.
 
 Tính năng
 
- **Player vs Player** — 2 người chơi chung 1 máy
- **Player vs AI** — 3 mức khó Easy / Medium / Hard
- **Admin Dashboard** — Thống kê, biểu đồ, CRUD 
- **Auth** — Đăng ký / Đăng nhập JWT, phân quyền Admin vs Player
 
Frontend SolidJS, Vite

Backend  Ruby on Rails 7 (API) 

Database PostgreSQL 

Auth JWT + BCrypt 

AI Minimax + Alpha-Beta (JS + Ruby)
